ClazzJS
=======

Portable JavaScript library for class-style OOP programming

Documentation is coming soon. Until that look at usage example: http://plnkr.co/edit/c5Xveb

На основной странице:
    - Общее описание
    - Разделы документации
    - Примеры

Разделы документации:
    - Installation (5%)
    - Creating of new clazz (20%)
        - Declaring a new clazz
        - Creating a new clazz
        - Creating new instances
    - Constants, Properties, Methods (40%)
        - properties
        - methods
        - constants
        - clazz_properties
        - clazz_methods
        - events (ссылка на Events)
        - clazz_events (ссылка на Events)
    - Inheritance (60%)
    - Events (80%)
    - Dependencies
    - Namespaces (100%)
